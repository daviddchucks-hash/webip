Network Tools

DNS propagation, traceroute, ping, port scan, WHOIS/RDAP and calculators — all running client-side via free CORS-enabled APIs.

🌍

DNS Propagation Checker

Compares the DNS answer returned by Google, Cloudflare and Quad9 resolvers — useful for checking whether a DNS change has propagated globally.

🛰️

Traceroute

Runs a server-side traceroute (via HackerTarget's free API — limited requests/day) and returns the hop list.

Results will appear here…
📶

Ping Test (HTTP Latency)

Browsers cannot send raw ICMP pings. This measures HTTPS round-trip latency instead — a practical proxy for reachability and responsiveness.

🔓

Port Scanner

⚠ Best-effort browser probe of common web ports only. Browsers block raw sockets and many ports (21, 22, 23, 25…) outright, so this cannot replace a real scanner like nmap. Only scan targets you own or are authorized to test.

📄

WHOIS / RDAP Lookup

Uses RDAP (the modern WHOIS replacement) — works for domains and IP addresses, free and CORS-enabled.
